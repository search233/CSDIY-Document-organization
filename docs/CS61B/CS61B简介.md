# CS61B 简介

CS61B 是由伯克利开设的 CS61 系列的第二门课程 ，课程聚焦于 **数据结构与算法设计**。



这门课程与 CS61A 比较相像，他有完善的文档、详细的教学视频、对应的课件等，同时也包含了一些非常有价值的 *lab*、*homework*、*project* 以及对应的评测系统。他的 2021 年春季的版本是完全开源的，其他版本的课程资料也是大部分开源的。



我学习的就是 2021 年春季版本。除了资源完全开源以外，这个版本提供了 [gradescope.com](https://www.gradescope.com/) 网站的邀请代码。在该网站上点击注册、作为学生注册，然后课程邀请代码填写 **MB7ZPY**，学校填写 *UC Berkeley* ~~（别傻乎乎的填 wust，找不到的）~~，再填写完名字和邮箱地址以后就可以在线评测你的作业了。作业可以直接打包为 *zip* 文件提交也可以直接关联自己的 *github* 账号，选择对应的仓库提交。



这门课程的作业是使用 Java 完成的，但是完全不用担心自己不会 Java。这门课程会在最开始的几周里，从面向对象从出发，一点一点带着你解决语言问题，如果有一定语言基础（不需要是 Java），语言学习会成为这门课最顺利的地方。同时这门课在最开始还会详细介绍 *IDEA* 的使用、*Unit Testing* 等等实用的东西。其中有一个 *project* 是让你去做一个 *gitlet* 。在项目的开始会给你详细的介绍 *git* 的使用逻辑，能够给你建立对于 *Commit*、*Branch*、*Blob*、*Staging Area* 这些概念的正确认识。
